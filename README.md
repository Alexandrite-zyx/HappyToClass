# 云开发 quickstart

## 这是HappyToClass的使用指南，使用前请确保拥有如下配置

- 微信开发者工具：云开发
- 后台：node.js

## 使用说明

- 确保部署小程序云开发
- 确保**app.js**文件中env配置为本地云开发环境ID
- 确保云数据库创建集合 
  - COURSE_LIST   
  - STUDENT_COURSE
  - STUDENT_LIST
  - TEACHER_LIST
- 在详情中开启不校验合法域名
